# Open Source Notices

DirectFast uses open-source software from the Flutter ecosystem.

This file documents the main third-party packages used in the project and why they are included. For full and current license texts, see each package page on pub.dev and the in-app license screen (Settings -> About -> Open Source Licenses).

## Main Dependencies

| Package | Purpose in DirectFast | Source |
|---|---|---|
| Flutter SDK | Cross-platform UI framework and runtime | https://github.com/flutter/flutter |
| flutter_riverpod | State management for app-wide feature state | https://pub.dev/packages/flutter_riverpod |
| riverpod_annotation / riverpod_generator | Riverpod code generation utilities | https://pub.dev/packages/riverpod_annotation |
| go_router | Declarative navigation and route management | https://pub.dev/packages/go_router |
| shared_preferences | Local persistence for settings/history/templates | https://pub.dev/packages/shared_preferences |
| url_launcher | Opening deep links and web/app URLs | https://pub.dev/packages/url_launcher |
| flutter_animate | UI animations and transitions | https://pub.dev/packages/flutter_animate |
| google_fonts | Google Fonts integration | https://pub.dev/packages/google_fonts |
| flutter_svg | SVG icon rendering | https://pub.dev/packages/flutter_svg |
| clipboard | Clipboard read helpers | https://pub.dev/packages/clipboard |
| intl | Localization helpers and formatting utilities | https://pub.dev/packages/intl |
| qr_flutter | QR code generation | https://pub.dev/packages/qr_flutter |
| uuid | Unique ID generation for local models | https://pub.dev/packages/uuid |
| share_plus | Native share sheet integration | https://pub.dev/packages/share_plus |
| gal | Saving media assets (QR export) to gallery | https://pub.dev/packages/gal |
| crypto | Hashing and cryptographic primitives | https://pub.dev/packages/crypto |
| encrypt | Local AES encryption/decryption utilities | https://pub.dev/packages/encrypt |
